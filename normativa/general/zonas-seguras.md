# ZONAS SEGURAS

---

## Definición

{% hint style="info" %}
**Concepto:** Las zonas seguras son áreas designadas donde queda **estrictamente prohibido** cualquier tipo de rol agresivo.
{% endhint %}

Estas áreas están destinadas a promover la **interacción pacífica** y la convivencia entre jugadores.

---

## Prohibiciones en Zonas Seguras

Dentro de las zonas seguras, está prohibido:

<table>
  <tr>
    <td>❌ Enfrentamientos físicos</td>
    <td>❌ Amenazas verbales</td>
  </tr>
  <tr>
    <td>❌ Uso de armas</td>
    <td>❌ Campeo</td>
  </tr>
  <tr>
    <td>❌ Reglaje</td>

  </tr>

{% hint style="danger" %}
**Importante:** El incumplimiento de esta norma será considerado una falta grave y podrá ser sancionado de forma inmediata por la administración.
{% endhint %}

## Lista de Zonas Seguras

### Zonas Protegidas

| Zona | Descripción |
|------|-------------|
| **Comisarías** | Todas las estaciones de policía |
| **Hospitales** | Centros médicos y EMS |
| **Bancos** | Entidades bancarias (salvo durante atraco activo) |
| **Garajes** | Todos los garajes públicos y privados |
| **Tiendas de Ropa** | Establecimientos de vestimenta |
| **Puntos de Spawn** | Áreas de aparición inicial |
| **Locaciones de Trabajos** | Dentro de los puntos de trabajo |

{% hint style="warning" %}
**Nota sobre Bancos:** Durante un atraco activo coordinado, el banco deja de ser zona segura temporalmente. Una vez finalizado el evento, vuelve a ser zona protegida.
{% endhint %}

## Excepciones

### Situaciones Especiales

Existen casos excepcionales donde el rol puede continuar en zonas normalmente seguras:

#### 1. Rol Previo en Curso

Si un rol activo (persecución, enfrentamiento) ya estaba en progreso **antes** de llegar a la zona segura:

* El rol puede **continuar brevemente** para concluir
* No se puede **iniciar** nuevo rol agresivo
* Se debe resolver rápidamente y salir

{% hint style="info" %}
**Ejemplo:** Si estás siendo perseguido y entras accidentalmente a un garaje, la policía puede seguir la persecución para detenerte, pero no puedes usarlo como refugio permanente.
{% endhint %}

#### 2. Autorización del Staff

En eventos especiales organizados por el staff, ciertas zonas pueden ser temporalmente habilitadas para rol agresivo con aviso previo.

## Consecuencias de Violación

### Sanciones

| Gravedad | Sanción |
|----------|---------|
| **Primera vez** | Advertencia + Posible revive de víctimas |
| **Reincidente** | Ban temporal (24-48 horas) |
| **Abuso repetido** | Ban extendido (7+ días) |
| **Extremo** | Ban permanente |

{% hint style="danger" %}
**Agravante:** Si el incumplimiento incluye RDM (matar sin razón) en zona segura, la sanción será más severa.
{% endhint %}

## Interpretación Correcta

### Lo que NO puedes hacer

* Esconderte en una zona segura tras cometer un crimen
* Usar zonas seguras como base de operaciones
* Provocar a otros jugadores desde zona segura
* Iniciar conflictos verbales que escalen fuera de la zona

### Lo que SÍ puedes hacer

* Socializar pacíficamente
* Realizar transacciones comerciales
* Hacer rol pasivo (conversar, trabajar)
* Reportar problemas al staff sin romper rol

---

## Realismo vs Mecánicas

{% hint style="success" %}
**Filosofía:** Aunque el servidor permite disparar mecánicamente en estas zonas, se espera que los usuarios actúen con sentido común y respeto a la normativa.
{% endhint %}

### Sentido Común

Piensa:
* ¿Dispararías dentro de una comisaría en la vida real?
* ¿Iniciarías una pelea en un hospital?
* ¿Sería lógico atacar a alguien en un garaje público lleno de gente?

**La respuesta es NO.** Respeta las zonas seguras para mantener el orden y el realismo.

---

## Reportes

Si presencias una violación a las zonas seguras:

1. **No respondas con violencia** (serías sancionado también)
2. **Captura evidencia** (clip/video)
3. **Reporta vía Discord** con la evidencia
4. Deja que el staff maneje la situación

{% hint style="info" %}
Los reportes falsos o malintencionados también serán sancionados.
{% endhint %}
